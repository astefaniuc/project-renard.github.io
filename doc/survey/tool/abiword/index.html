AbiWord is a word processor. It is written using Gtk+3 and provides an embeddable widget called AbiWidget that can be loaded into other Gtk+3 applications.

It can be extended with plugins that can extend functionality. These can be used to add more import/export file formats.

Description

  • Screenshots of the GUI
  • Spell-checking can use aspell, ispell, or enchant
  • Menus are described in the manual
  • Bidirectional text support
  • AbiSource also hosts several related projects.
    • In particular, the Link Grammar natural langage parser.
    • An unmaintained library for pen input: libstylus
  • Collaborative editing: AbiCollab
