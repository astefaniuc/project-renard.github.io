Wikipedia (en)

Paperpile is a reference management software that is distributed as a Chrome extension. It is tightly integrated with Google Apps services such as Google Docs and Google Drive. This allows for easy cloud storage and sharing of bibliography information with collaborators from inside a real-time collaborative editor.
