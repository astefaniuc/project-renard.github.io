Curie GUI displaying the document properties of Social Determinants and universal health coverage by Professor Sir Michael Marmot from Determinantes de la salud. See also Marmot, M. (2013). Universal health coverage and social determinants of health. The Lancet.

The fourth pre-alpha release of Curie is now available. It features the following user-visible improvements:

  • The File -> Properties menu displays the document properties of the currently open PDF document.
